# 欢迎来到`RobbotRews`

## 介绍

##### 这是什么？

这是一款**自动化+生存类游戏**, 一款**完全免费**的游戏，在这款游戏中***你将是一个机器人***,被派到一个`星球🌕`中采取数据,在星球上你要**时刻注意自己的方方面面**从而***生存*** 下去,你不断开采资源，建造传送带，制造出更多`🤖机器人`，通过`编程`来控制这些机器人**为你干活**,随着不断对`星球`探索的深入，你发现了星球上的**原住民🧍‍♂️**,你可以为他们制造🏠家园，或者🦵逐出，你可以通过**千奇百怪的方式制造**些小东西🛡️，你也可以**制造机器配件**通过**编程为**自己设计配件🦾🦿🧠，在这途中你要躲过调查的`监视机器人👁️`和`各种各样奇特的生物🧌`，但注意你的每一个行为都会影响到`你自己`，和`星球的变化`。
